### 🚀 Projects & Contributions

#### **1. Mall Connect**
A feature-rich mall management and shopping experience platform.
- **Backend**: Built with **NestJS**, utilizing WebSockets for real-time communication and TypeORM for complex database management.
- **Frontend**: Developed with **Next.js**, featuring interactive **3D floor maps** using **React Three Fiber** and Three.js.
- **Features**: Includes real-time chat between users and store managers, multi-store management, and an administrative dashboard for analytics and activity oversight.

#### **2. Reqto Platform**
A specialized request and workflow management system.
- **Functionality**: Automates the submission, approval, and tracking of service requests within an organization.
- **Architecture**: Designed with a focus on role-based access control and high availability, ensuring streamlined operations for both administrators and end-users.

#### **3. Unified Headless Video Core**
A high-performance logic engine for cross-platform video playback.
- **Technology**: Written in **Rust** to ensure memory safety and maximum performance.
- **Core Concept**: Implements a "headless" architecture, meaning the core logic is independent of the UI, allowing it to be used as a backend library for mobile, desktop, and web applications.

#### **4. CrosswordApp**
An interactive crossword puzzle game for mobile and web.
- **Mobile**: Built using **Flutter** for a smooth, native cross-platform experience.
- **Design**: Focuses on engaging user interactions, progress tracking, and custom game board generation logic.

#### **5. ShiftMate**
A project and resource management tool tailored for team scheduling.
- **Core Features**: Manages employee shifts, resource allocation, and project timelines. Includes automated conflict detection and a centralized calendar for team visibility.
